Περιγραφή:

Ολοκλήρωση της λειτουργικότητας της σελίδας των 2 πρώτων εργασιών ως πλήρη εφαρμογή ιστού, υλοποιώντας τα ακόλουθα:

  • Αρχικό setup δήλωσης σταθερών στοιχείων πίνακα δεδομένων (π.χ. αγώνισμα 100m, 5km, πλήθος αγωνιζομένων, τύποι στηλών πίνακα κ.λπ.).
  • Δημιουργία PWA: ανάρτηση στο github (ή οποιοδήποτε https secure context site), δημιουργία manifest και service workers, έτσι ώστε να μπορεί να γίνει "εγκατάστάσιμο" (installable). Αποθήκευση όλων των δεδομένων στο localStorage
  • Άλλες προαιρετικές ιδέες μόνον αν σας εξπηρετούν:
    • Επικοινωνία με οποιαδήποτε μορφή backend
    • Δημιουργία γραφημάτων
    • Τήρηση πολλαπλών πινάκων αθλημάτων
